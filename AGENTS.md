# Codex 공동 작업 안내

이 저장소는 한국 고등학교 수학교사를 위한 오프라인·정적 웹앱 **수학 수행평가 세특 메이커**이다.

## Codex가 작업을 시작할 때

1. 저장소 루트의 `CURRENT_WORK.md`를 먼저 읽는다.
2. 현재 파일과 Git 변경 상태를 확인한 뒤 기존 작업을 이어간다.
3. 사용자는 바이브코딩 초보이므로 결과 중심의 쉬운 한국어로 설명한다.

## 작업 원칙

- 반드시 이 저장소의 실제 파일을 직접 수정하고 검증한다.
- 기존 기능과 사용자의 변경사항을 보존한다.
- 정적 웹앱이며 별도 서버나 빌드 과정 없이 GitHub Pages로 배포한다.
- 앱 데이터는 브라우저 `localStorage`에 저장되므로 기기 간 자동 동기화되지 않는다.
- 사용자가 요청하지 않은 커밋, 푸시, 저장소 설정 변경은 하지 않는다.
- 작업을 마칠 때 `CURRENT_WORK.md`의 최근 작업, 현재 상태, 다음 할 일을 갱신한다.

## 저장소 및 배포

- GitHub 저장소: `https://github.com/chunoring/math-record-maker-.git`
- 배포 주소: `https://chunoring.github.io/math-record-maker-/`
- 기본 브랜치: `main`
- 사용자는 수정 후 GitHub Desktop에서 **Commit to main → Push origin**으로 배포한다.

## 주요 구현 기능

- 수행평가 등록·수정·삭제 및 localStorage 저장
- 9개 수학 과목과 163개 성취기준
- 학생 관찰 근거, 진로 계열, 교사 평가, 다중 수학 역량, 성취수준, 분량 설정
- 관찰 근거 우선의 세특 초안 생성 및 복사
- 성취기준 코드와 원문이 결과 문장에 직접 드러나지 않도록 처리
- 수행평가별 AI 프롬프트 복사와 사용방법 안내 모달
- 홈 화면 4단계 흐름도와 © 인창고 수학왕 배지
- 수행평가 JSON 백업 내보내기·불러오기 및 기존 목록과 병합
